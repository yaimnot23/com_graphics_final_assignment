최종 과제 구성 가이드

* 주제 자유
* 채점 기준 - 기획(20), 완성도(20), GI 기술 (DDGI or SurfelGI) 적용(20), 리포트(40)
* 리포트* 강의에서 배운 내용과 구현 내용 매핑
  * 게임 내용 및 개발 상세 설명
  * 모든 내용에 게임에서 작동하는 그림 첨부하여 설명 (필수)
* 중요* 게임 구동 안 되면 0점
  * 게임의 경우 git 에 올리고, 웹으로 구동되게 하여 링크 제출
  * 리포트 0점이면 전체 0점 처리
  * 리포트의 경우, 모든 개별 내용에 대해 구현한 게임에서 캡쳐한 이미지로 설명 (본인 게임 캡쳐 이미지로 설명 안 한 것은 모두 0점 처리)
  * 리포트 포맷은 MD 로 작성
  * 리포트는 해당 MD 가 오픈되는 git 링크로 제출할 것
* 기한* 채점은 6월 22일 (오전 9시~) 진행. random 순서로 제출한 링크에 접속하여 진행.
  * 링크 제출은 6월 21일 23시 59분까지 KLAS 에 제출.
  * KLAS 에 링크 제출 안 한 경우 0점 처리
  * KLAS 에 제출한 링크가 채점 시 안 열리면 0점 처리
  * 제출한 링크와 다른 링크 허용 X
